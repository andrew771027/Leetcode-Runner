# v3.1 — Result Model & Observability

---

# 當時的目標

讓 Execution 的結果有一致的格式。

---

# 為什麼會有這一版

v3 已經完成：

- Execution Backend
- Orchestrator
- Backend Abstraction

但開始遇到新的問題。

不同 Backend 回傳的結果格式不一致。

例如：

subprocess：

```python
CompletedProcess(...)
```

未來如果是：

- Docker Backend
- CI Backend
- Remote Backend

結果可能完全不同。

---
